# ビルドと修正

TypeScriptとビルドエラーを段階的に修正:

1. ビルドを実行: npm run build または pnpm build

2. エラー出力を解析:
   - ファイルごとにグループ化
   - 重大度で並べ替え

3. 各エラーについて:
   - エラーコンテキストを表示（前後5行）
   - 問題を説明
   - 修正を提案
   - 修正を適用
   - ビルドを再実行
   - エラーが解決したか確認

4. 停止条件:
   - 修正が新しいエラーを導入
   - 同じエラーが3回試行しても続く
   - ユーザーが一時停止を要求

5. サマリーを表示:
   - 修正されたエラー
   - 残りのエラー
   - 導入された新しいエラー

安全のために一度に1つのエラーを修正してください！
